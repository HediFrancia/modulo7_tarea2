# Proyecto Clases - Séptimo Módulo

Este proyecto agrupa las prácticas desarrolladas en el módulo 7.

## 🚀 Descripción del Proyecto

Este repositorio tiene como objetivo consolidar los conocimientos adquiridos en el módulo: Sistema de Gestión del Colegio San Marcos.

### 🛠️ Tecnologías Utilizadas

* **Lenguaje:** jsx, css, html
* **Frameworks/Librerías:** react, vite, axios
* **Otras herramientas:** git

---

## 📋 Características Principales

* Crear, editar alumnos: nombre, apellidos, grado y sección. 
* Envio de mensajes
* Visualizar si los alumnos están activos o no
